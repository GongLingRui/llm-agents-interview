# 网页总结：“懒人提示词”（两步提示法）技巧解析
## 核心观点颠覆
随着AI模型能力增强，**最有效、高产且易上手的提示词往往是简单表述**（需搭配推理模型），打破“提示词越详细越好”的传统认知，无需复杂框架（如PAIN、STAR），小白也能高效使用AI。

## 关键技巧：两步提示法（原称“Lazy Prompt”）
### 核心逻辑
|类型|核心差异|类比案例|
| ---- | ---- | ---- |
|传统提示词|用户明确告诉AI“怎么做”|“你写代码实现用户登录功能”|
|Lazy Prompt（两步提示法）|用户仅告知AI“要什么结果”，让AI自主决定“怎么做”|“你提用户登录功能的需求，再写方案+代码”|

### 具体操作
1. **第一步：让AI优化初始提示词**  
   输入格式：`优化提示词：「你的口语化需求」`  
   示例：需求为“输出一篇爆款微信公众号文章”，输入`优化提示词：输出一篇爆款微信公众号文章`，AI会自动生成结构化提示词，涵盖角色目标（5年+公众号撰稿人，目标10万+阅读）、主题价值（碎片化时间提升职场技能，核心是“精力颗粒度管理”）、结构风格（5个标题备选、5部分文章结构、真诚启发语气）、输出要求（2500-3000字+配图描述词）。
2. **第二步（可选）：让AI自我优化**  
   可补充提示词进一步提升质量，如：  
   - `请你自我优化，充分运用批评与自我批评这个武器，然后给出更好的版本`  
   - 专业版：`请你不要执行接下来的任务。你现在的身份是世界顶级的提示工程专家，请仔细阅读我提供的【初始提示词】，并从清晰度、专业度、结构化、模型适应性四个维度进行批判性优化。请仅输出优化后的提示词内容，并用 ``` 包裹起来`

### 适用场景
覆盖几乎所有“创作类”“结构化输出”任务，包括但不限于：写文章、论文、脚本、帖子、计划书、商业报告、运营方案、PPT大纲、课程设计、产品文案、故事/营销文案、数据分析结论。


## 延伸应用：图像生成领域
### 操作逻辑
输入提示词让AI生成图像的JSON格式结构化描述，示例提示词：`详细描述完整的图像复刻 JSON 提示词，包含物体、服装、头发、细节、配饰、摄像设备、环境、灯光、风格、身体动态，一切都要详细复刻原图，最终输出一个优化的元提示词， 800 字`。

### JSON提示词核心维度
包含画布（比例3:4、全身居中构图、镜面自拍）、主体（18-24岁东亚女性、身材/姿势）、发型（黑色高双马尾+熊猫发夹）、服饰（内搭白衬衫+外搭深海军蓝毛衣+格子百褶裙等）、配饰（深咖啡棕单肩包、香槟金手机）、环境（极简室内、冷灰墙+暖橡木地板）、光线（左侧柔和漫射日光）、相机（24-28mm焦距、平视角度）、色彩搭配、身体动态、构图规则、负面要求（不暴露面部、无额外家具等），可基于此调整生成不同风格图像（如运动风、通勤风穿搭）。
